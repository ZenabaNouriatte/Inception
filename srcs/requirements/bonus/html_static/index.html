Projet Inception

Inception est un projet de l'école 42 visant à faire découvrir les bases de la virtualisation avec Docker. Le but est de construire une architecture de containers cohérente et fonctionnelle.

Objectifs du projet

  • Configurer Docker Compose pour lancer plusieurs services
  • Construire vos propres images Docker
  • Mettre en place des services comme WordPress, MariaDB, Nginx, Adminer, etc.
  • Gérer les volumes, les réseaux, et les fichiers de configuration
  • Ajouter des bonus (site statique, Redis, FTP...)

Bonus actuel

Ce site est un conteneur bonus qui affiche une simple page HTML expliquant le projet.

Structure du projet

Les services sont définis dans le fichier docker-compose.yml et construits à partir de Dockerfile personnalisés dans srcs/requirements/.
